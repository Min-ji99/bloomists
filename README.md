# SEED FACTORY
<img width="484" alt="스크린샷 2023-04-10 오전 2 33 20" src="https://user-images.githubusercontent.com/61926751/230787727-919417ba-7b50-4b0b-8860-51152fec5f7c.png">

아이들의 교육을 위한 인터렉티브 컨텐츠로 자아가 성장하는 과정을 식물의 성장으로 표현한 AR 프로젝트입니다. 
가상의 오브젝트인 씨앗과 씨앗 공장, 실제 오브젝트인 빛, 물, 포션을 주며 씨앗을 성장시켜 나갈 수 있습니다.

# Tools
개발 : Unity, ARCore, Arduino
디자인 : 3D max, Maya

# Ideation
삶 속에서 있는 행복, 즐거움, 고난들을 공장을 통해 표현했습니다. AR 기술과 아두이노를 접목 시켜 블루투스 센서를 통해 빛 센서, 물을 인식해 가상 물체인 씨앗의 성장 과정을 직관적으로 볼 수 있습니다.
